# 🏛️ Cinema AI Studio Architecture

Version: 0.1 Alpha

---

# Purpose

This document defines the high-level architecture of the Cinema AI Studio project.

The architecture is designed to separate knowledge, decision making and execution.

---

# Core Philosophy

The system does not generate prompts.

The system generates cinematic decisions.

# Main Components

## Project Analyzer

Responsible for understanding the project.

Input:

- Theme
- Duration
- Audience
- Objective

Output:

- Project Profile

---

## Decision Engine

Responsible for choosing the best cinematic decisions.

Output:

- Emotion
- Narrative
- Camera
- Lighting
- Art
- Sound
- Editing

---

## Blueprint Generator

Creates the complete cinematic blueprint.

---

## Prompt Translator

Transforms the blueprint into prompts compatible with AI video generators.
